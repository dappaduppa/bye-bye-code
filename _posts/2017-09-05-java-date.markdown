**Don't get mislead by variable names**
```java
final LocalDate today = LocalDate.now();
System.out.println(today);
final LocalDate tomorrow = today.plus(-1, ChronoUnit.DAYS);
System.out.println(tomorrow);
final LocalDate actToday = tomorrow.minus(1, ChronoUnit.DAYS);
System.out.println(actToday);
/* Output (executed on 2017/09/06)
 * 2017-09-06
 * 2017-09-05
 * 2017-09-04
 */
```
