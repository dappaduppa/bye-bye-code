```vi
remove empty lines
------------------

1) 
:v/./d

   v --> reverse global

2) 
:g/^$/d
   

3) leave at least one empty line
:%!cat -S 

	or
	
:%!cat -r


4) 

vim
----
:g/^\s*$/d
^ begin of a line
\s* at least 0 spaces and as many as possible (greedy)
$ end of a line


vi \s won't work
--
:g/^ *$/d
```


## palindrome
```re

sublime 5 letter palindrome  
([a-z])([a-z])([a-z])\2\1

in vim  
\([a-z]\)\([a-z]\)\([a-z]\)\2\1


sublime match 3 similar chars  
([a-z])\1{2}

in vim  
\([a-z]\)\1\{2\}

```
